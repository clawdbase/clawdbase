# Changelog

All notable changes to clawdbase will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Terminal UI with real-time portfolio dashboard
- Clawd AI integration for trade signal analysis
- Comprehensive test suite with unit and integration tests
- CI/CD pipeline with GitHub Actions
- CLI commands for manual trading operations

### Changed
- Improved error handling throughout
- Better logging with structured output
- Refactored trading engine for extensibility

### Fixed
- Rate limiting handling for API calls
- Order execution edge cases

## [0.3.0] - 2024-01-10

### Added
- Trading engine with strategy support
- DCA (Dollar Cost Averaging) strategy
- Momentum strategy
- Mean Reversion strategy
- CLI interface for quick commands
- Portfolio service for balance tracking
- Order service for trade execution

### Changed
- Improved configuration system
- Better type definitions

## [0.2.0] - 2024-01-05

### Added
- Coinbase Advanced Trade API client
- Authentication with HMAC signatures
- Account fetching
- Price/ticker data
- Product information

### Changed
- Project restructure to proper modules
- TypeScript configuration improvements

## [0.1.0] - 2024-01-01

### Added
- Initial project scaffold
- TypeScript configuration
- Basic type definitions
- Logger utility
- Environment validation

## [0.0.1] - 2023-12-28

### Added
- Initial commit
- Basic project structure
