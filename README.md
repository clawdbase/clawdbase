# clawdbase

Coinbase trading terminal powered by Clawd AI.

## Setup

```bash
npm install
cp .env.example .env
# Add your Coinbase API credentials
npm run build
npm start
```

## Development

```bash
npm run dev    # Watch mode
npm run lint   # Check code
```
